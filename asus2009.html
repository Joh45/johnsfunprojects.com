John's Fun Projects - ASUS Pasively cooled PC

Project Overview

Asus 2009 windows 7 PC

This is the computer I bought in 2009. It is probably the best computer I have ever had. Bought it from best buy but slowly added parts and modified it to be much more powerful and also passively cooled. It is mostly fanless. It has a Fanless, passively cooled GTX 750 GPU. It has 2 large HDDs in it and it stores a lot of files and has a lot of ports. I plan to clone its main hard drive on a large ssd. It has a lot of classic games on it, and there's still some situations where it can do things that my laptop can't.

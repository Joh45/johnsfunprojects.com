My build order website

Last year, i copied the html of a famous aoe2 build order site called buildorderreference.com so that I would have a nice place to write down my own build orders. I believe less than 5 people will ever possibly see that I did this, so I don't think I will ever have to worry about copying the work of cicero, the original author of the site. If somehow more people ever do look at it, i'll convert it into a new site at that point. Anyway, My site is called Joh's Build Order Reference.
